### 一、介绍

基于springboot的爱心捐赠管理系统

开发语言：java

运行环境:idea或eclipse 数据库:mysql

技术：springboot+mybatis+html+layui+echarts

爱心捐赠系统（用户端+管理端）

用户端功能模块：登录+注册+衣物捐赠+捐赠浏览+论坛交流+帖子留言+爱心许愿+个人主页

管理端功能模块：登录+用户管理+捐赠记录管理+论坛管理+留言管理+心愿管理
